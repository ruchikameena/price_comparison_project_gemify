### Getting Started with CRA project - gemify

This project is build to address very common problem where we have to visit on different portals to compare products here we provide You an platform where we list products from various platform on our gemify portal, give you an easy to understand price comparision platform.

## Tech Stack used :-
-> React.js - for frontend
-> Firebase - for authentication
-> Mongodb - for database
-> Express.js/Node.js - for backend
-> Flask - for chatbot
-> python - ML and Api creation (i.e. data fetching from various platforms)

## My Role here:-

To design & develop an user friendly UI/UX application consisting of - Products Page, Chatbot design & integration, backend integration for data fetching.
